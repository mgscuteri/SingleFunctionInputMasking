Single Function Input Masking

This page demonstrates how to perform input masking on HTML input objects by binding a single stateless javascript function to the "input" event of all input objects containing a value for the optional "data-mask" HTML attribute.

See functioning demo on codepen here

Driving HTML Properties
"data-format" property uses predefined aliases that map to regEx strings in the "regExes" array
"data-fomrat" controls what displays in the input field by default
(These two values must be the same length)

Already Implemented RegEx aliases
alphabetical = 'a'
alphaNumeric = 'A'
numeric = *
Custom = (Feel free to add additional regEx aliases to "regExes" array)
